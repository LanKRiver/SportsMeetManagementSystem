# Sports Meet Management System
校运动会管理系统 | 运动会智能排程与成绩管理

## 项目介绍
这是一个基于 C++ 开发的校园运动会管理系统，实现运动员报名、智能秩序册生成、成绩录入、排名计算、总分统计等完整功能。

系统支持自动安排比赛时间、处理运动员项目冲突、分男女组别与参赛项目分组，满足学校运动会的实际使用需求。

## 功能清单
1. 运动员报名管理（姓名、性别、学院、参赛项目）
2. 比赛项目与规则查看
3. 智能自动生成运动会秩序册
   - 遵守跑道/场地使用规则
   - 径赛串行、田赛三场地并行
   - 自动避开午休时间（12:00-14:00）
   - 超过6人自动分组
   - 一天无法完成自动延长至第二天
4. 运动员时间冲突检测与提示
5. 成绩录入与格式自动转换（秒/分秒/距离）
6. 自动排名与计分
7. 个人总分查询
8. 数据持久化保存与清空

## 开发环境
- 编程语言：C++
- 开发工具：Visual Studio / Dev-C++ / CLion
- 运行系统：Windows

## 文件结构
- main.cpp                  系统主入口与菜单
- sport.h / sport.cpp        运动员数据结构定义
- signup.h / signup.cpp      报名系统
- event.h / event.cpp        项目信息与时间格式转换
- grade.h / grade.cpp        成绩录入
- rank.h / rank.cpp          排名与计分
- schedule.h / schedule.cpp  智能排程与秩序册
- fileIO.h / fileIO.cpp      文件保存与读取

## 使用说明
1. 编译运行 main.cpp
2. 按照菜单提示依次使用：
   1 选手报名
   2 项目查询
   3 生成秩序册
   4 查看秩序册
   5 成绩录入
   6 排名公布
   7 个人总分
   8 清空数据
   9 保存退出

## 项目特点
- 纯控制台输出，无特殊字符，兼容性强
- 智能时间安排算法，符合真实运动会场地与时间规则
- 自动处理运动员参赛冲突
- 支持长时间项目自动顺延至次日
- 数据自动保存到文件，重启不丢失
